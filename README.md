# PokemonApp

## Instalación

1. Clonar el repositorio
2. Clonar el archivo .env.template y renombrarlo a .env
3. Ejecutar el comando `npm install` para instalar las dependencias

## Uso

1. Ejecutar el comando `npm run dev`
